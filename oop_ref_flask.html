API Design: An example in Flask (week 12)

    Flask is a web development library in Python (Flask Documentation, N.D.) which uses a certain framework (templates, python file and a database). The package that you will find when you click on the button below therefore contains a python application file (“application.py”), several html templates and a sql database file (“students.sql”) and it is only a short introduction to Flask. For running the program, you still need to have an available URL address, which is not provided here. If you have one, you will see that the program is very simple. The related database contains information about grades for students for different courses and the app allows to search for the grades of all students by a specific course.
